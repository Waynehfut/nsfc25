## Abstract

微创手术是普外科、心胸外科等二十余个科室的重要诊疗手段，开展微创手术质量控制是保障人民群众生命财产安全的重要途径，更是国家推进“健康中国2030”战略的重要组成之一。针对手术质量控制中的精确响应困难、多模数据处理和质控过程复杂等问题，本项目采取数据驱动与行为建模相结合的思路研究智能化微创手术质量控制方法，基于结构化、文本型、影像型等微创手术多模态数据，探索融合专家知识引导与特征语义关联的手术特征发现方法，设计结合操作行为型微观尺度、过程阶段型中观尺度、流程周期型宏观尺度的手术行为理解与异常感知机制，建立基于多属性决策机制和关键评价指标响应的过程质量综合评价与反馈体系，并以前期搭建的“德医云”微创医疗服务平台为案例开展验证研究。项目研究将为解决行为响应的手术质量控制问题提供新的方法和途径，有助于微创手术智能决策理论体系的丰富和优质医疗资源与服务的公平可及，具有重要的理论意义和实际价值。
</HighlightedSection>

## Visuals

### Picture

Use the `Picture` component to display images. If you have a figure in PDF format, you can pass in its path as the `src` prop and it will be converted into a web-friendly image automatically.

<Picture src={dogsDiffc} alt="Photo of two running dogs, lossily compressed using the DiffC algorithm" />


### Tab panels

Tab panels are a great way to display a labeled collection of related visuals without taking up too much space.

<Tabs defaultValue="true">
  <TabsList>
    <TabsTrigger value="true">Ground truth</TabsTrigger>
    <TabsTrigger value="diffc">DiffC (Flux-dev)</TabsTrigger>
    <TabsTrigger value="msillm">MS-ILLM</TabsTrigger>
  </TabsList>
  <TabsContent value="true">
    <Picture
      src={dogsTrue}
      alt="True photo of two dogs running side-by-side in shallow water"
    />
  </TabsContent>
  <TabsContent value="diffc">
    <Picture
      src={dogsDiffc}
      alt="Photo of two running dogs, lossily compressed using the DiffC algorithm"
    />
  </TabsContent>
  <TabsContent value="msillm">
    <Picture
      src={dogsMsillm}
      alt="Photo of two running dogs, lossily compressed using MS-ILLM"
    />
  </TabsContent>
</Tabs>
